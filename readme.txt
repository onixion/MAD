	Verzeichnisse:
		doc ... Dokumentation über ein paar Programmabschnitte und Notizen
		lib ... Bibliotheken für das Programm
		src ... Sourcecode des Programmes

	Installation:
		Windows: Projekt mit Visual Studio (C#) öffnen und kompilieren
		Linux: Mono und Monodevelop installieren; src/MAD.proj mit Monodevelop öffnen und kompilieren
		
		(Hinweis: Programm braucht Root rechte zum starten.)

	Programme:
		src/MAD 	... Hauptprogramm, CLI, CLI-Server, GUI, Jobsystem, Datenbank
		src/CLIClient 	... CLIClient
		src/MadNet 	... eigene Bibliothek für das Empfangen und Senden von Paketen, AES
		src/CLIIO 	... eigene Bibliothek für das Lesen und Schreiben auf die Konsole
		src/SmartThreadPool ... SmartThreadPool ('https://github.com/amibar/SmartThreadPool')
		
	(Dieses Verzeichnisse sind auch online unter 'https://www.github.com/onixion/MAD.git' erreichbar.)
